  1. Home
  2. QR Generator

Project information

  • Category: Web Application
  • Client: Final Year Project
  • Project date: 01 March, 2023
  • Technologies: MERN Stack
  • Contribution To: Associated with Elzian Agro Internal Project

Key Features

QR Generator is an application developed using the MERN stack to support the Elzian Agro Earth Restoration Project. Registered users involved in earth restoration can log in to the system and upload their environmental data. The web application then generates a QR code based on the provided details, which users can share with clients.
